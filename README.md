# Axel Acosta | Portfolio



## About This Project

This isn't a template. It’s a handcrafted developer portfolio built with the same engineering principles I apply to backend systems: **separation of concerns, modularity, and maintainability.**

- **Architecture first**: Sections, data, and logic are decoupled.
- **Real i18n**: Manual, library-free English/Spanish translation system with smooth transitions.
- **Design with purpose**: Modern, accessible UI using Tailwind CSS.

## Tech Stack

| Category | Technology |
|----------|------------|
| **Framework** | Astro |
| **Styling** | Tailwind CSS v4 |
| **Language** | TypeScript |
| **Hosting** | Vercel |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Axllacosta/portfolio-v1.git
cd portfolio-v1

# Install dependencies
npm install

# Start the dev server
npm run dev
```
